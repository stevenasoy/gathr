# Vocabulary

Use these exact role names in UI copy.

## Roles

- **Host** — the supply side; lists and manages venues.
  - UI: "Become a Host", "Host dashboard", "message the Host", "For Hosts", "Host resources"
- **Gatherer** — the demand side; books a venue to gather their people (the brand pun on "Gathr").
  - Use "Gatherer(s)" in copy that refers to the booker, not "guest/organizer/renter/traveler"

## Mode labels

- The Host workspace shows **"Hosting"**
- The toggle back to the booking experience reads **"Switch to gathering"**
- Internal mode values in `ModeContext`/localStorage are still `'hosting'`/`'traveling'` — not user-visible; don't rename them

## What NOT to use

Do **not** use Airbnb's "traveler/traveling" or generic "venue owner / organizer" in UI copy — the canonical pair is **Host ↔ Gatherer**.
